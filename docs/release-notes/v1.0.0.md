# v1.0.0 — Initial Release

U.GG Memory Wrangler is a small Windows utility that restarts [U.GG](https://u.gg) to reclaim memory that accumulates during long play sessions. Double-click to run — no installation, no configuration required.

---

## Features

- **One-click restart** — double-click the `.exe` while U.GG is running and it handles everything automatically
- **Live progress window** — a native WPF window appears showing each step (stopping, starting, settling) as it happens in real time
- **Memory reporting** — measures Working Set – Private before and after restart (matching Task Manager) and shows exactly how much was freed
- **Lifetime tracking** — persists a running total of memory freed and average per restart across sessions
- **Toast notifications** — fires a Windows notification with the result summary when done
- **Auto-detection** — finds U.GG's install path automatically from the running process; falls back to common install directories and saved config if U.GG isn't currently open
- **Path memory** — saves the resolved exe path so subsequent runs work even if U.GG hasn't been launched yet
- **CLI mode** — full flag-based control for advanced users and scripting (`-name`, `-path`, `-delay`, `-warmup`, `-kill-only`, `-track-memory`, and more)
- **No dependencies** — single `.exe`, no runtime, no installer, no admin rights required

---

## Installation

1. Download `ugg-memory-wrangler.exe` from the [latest release](https://github.com/kevinslawinski/ugg-memory-wrangler/releases/latest)
2. Place it anywhere — Desktop, a tools folder, wherever is convenient
3. Double-click while U.GG is running

> **Windows SmartScreen warning:** This executable is unsigned. Windows may show "Windows protected your PC" on first run. Click **More info → Run anyway** to proceed. Alternatively, [build from source](../../README.md#option-2--build-from-source) to skip SmartScreen entirely.

### Verify the download (optional)

Compare the SHA-256 hash of your downloaded file against `checksums.txt` in the release assets:

A. On the asset download page next to the .exe file name

B. Using PowerShell:

```powershell
(Get-FileHash ugg-memory-wrangler.exe -Algorithm SHA256).Hash.ToLower()
```

## Build from source

```powershell
git clone https://github.com/kevinslawinski/ugg-memory-wrangler.git
cd ugg-memory-wrangler
go build -ldflags="-H windowsgui -s -w" -o ugg-memory-wrangler.exe .
```

Requires [Go 1.22+](https://go.dev/dl/). See the [README](../../README.md) for full usage and CLI flag documentation.
